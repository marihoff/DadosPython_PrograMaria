# DadosPython_PrograMaria

Um repositório com exercícios e exemplos de Python do curso PrograMaria, com foco em programação básica e análise de dados.

## 📚 Sobre

Este projeto reúne arquivos de aulas em Python que acompanham o conteúdo do curso. Cada script representa um exercício ou exemplo de uma aula, permitindo estudar conceitos de Python de forma prática.

## 🗂️ Estrutura do repositório

- `aula01.py` até `aula32.py`: scripts das aulas de Python.
- `aula6.py`, `aula8.py`, `aula9.py`: arquivos adicionais de aulas.
- `analise_dados_mod7_(1).xlsx` e `planilha_modulo3.xlsx`: planilhas usadas em exercícios de análise de dados.
- `PROGRAMARIA PYTHON DADOS/`: pasta com outra cópia ou versões alternativas das aulas.

## 🚀 Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/marihoff/DadosPython_PrograMaria.git
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd "D:\PROGRAMARIA PYTHON DADOS"
   ```
3. Execute um dos scripts com Python:
   ```bash
   python aula23.py
   ```

## ✅ Requisitos

- Python 3.8 ou superior
- Um editor de código ou terminal de sua preferência

## 💡 Dicas

- Abra os arquivos `.py` para revisar os exemplos e modificar os trechos conforme necessário.
- Use as planilhas `*.xlsx` para praticar importação e análise de dados no Python.
